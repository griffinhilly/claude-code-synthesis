# Session Summary Template

Output this format:

```
SESSION COMPLETE — [Project Name]

ACCOMPLISHED:
- [Key outcome 1]
- [Key outcome 2]
- [Key outcome 3]

SESSION REVIEW:
[Reviewer's assessment — presented directly, not filtered]
[Reviewer's suggestions]

SKILL HEALTH:
[Which skills fired, any underperformance or gotchas updates]

EXECUTION GAPS:
[Any CLAUDE.md instructions triggered but not followed, or "none detected"]

CROSS-PROJECT:
[Transferable learnings, or "none this session"]

COMP UPDATES:
- PLAN.md: [updated / no changes]
- MEMORY.md: [updated / no changes]
- ORIENT.md: [updated / no changes]
- CLAUDE.md: [updated / no changes]

BLOAT CHECK: [all clear / MEMORY.md at N lines, consider pruning]

NEXT SESSION: [What PLAN.md says is next]
```

Keep it concise. The summary is a dashboard, not a narrative.
