# Session Review Guide

Dispatch a **Session Reviewer** agent using the delegation template from `~/.claude/guides/delegation-templates.md`.

## What the reviewer receives
- **Session notes** from Step 1
- **PLAN.md** current state section (what was the goal coming in?)
- **MEMORY.md** (relevant accumulated knowledge)

## What the reviewer evaluates
From clean context (no shared assumptions with the orchestrator):
- What was actually delivered vs. what was planned
- Quality assessment (evidence-based, not vibes)
- What was overlooked or deferred without good reason
- Patterns (recurring issues, if visible across session notes)
- 1-3 specific suggestions for future sessions

## Presentation Rule
The reviewer's output is for the user, not for the orchestrator. Present it directly — do not filter, summarize, or editorialize the reviewer's findings. If the reviewer says something uncomfortable, that's the point.
